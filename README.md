## Introduction

This repository contains the source code for a high-performance, responsive personal website built using HTML5 and CSS3. The primary goal of this project was to implement advanced styling techniques—such as Flexbox layouts, linear gradients, and media queries—to create a seamless user experience across all devices. This project represents a key milestone in my journey toward becoming a Professional Software Developer at ECPI University.

## Technologies Used
* **HTML5:** Semantic tags (`<nav>`, `<header>`, `<main>`, `<section>`) and structured data tables.
* **CSS3:** Flexbox, Media Queries, Linear Gradients, and Transitions.
* **XAMPP:** Local server hosting via Apache.
* **GitHub:** Version control and GitHub Pages for live deployment.

## Features

### 1. Semantic HTML5 Architecture
The site is built using a clear, accessible document hierarchy. By using tags like `<header>`, `<nav>`, `<main>`, and `<section>`, the project ensures better SEO and screen-reader compatibility.

### 2. Multi-Level Flexbox Layouts
* **Navigation:** Implemented a horizontal navigation bar using `display: flex` and `justify-content: center` for a clean, centered look.
* **Contact Form:** Used a vertical Flexbox stack (`flex-direction: column`) to ensure form inputs and labels align perfectly on all screen sizes.

### 3. Interactive UI & Micro-interactions
* **Smooth Transitions:** Applied `transition: all 0.3s ease` to buttons and links, creating a professional feel when users hover over interactive elements.
* **Dynamic Scaling:** Added a `transform: scale(1.02)` effect to main sections on hover, providing subtle visual feedback to the user.
* **Modern Aesthetics:** Utilized a 135-degree linear gradient background (`#f5f7fa` to `#c3cfe2`) for a high-end, contemporary appearance.

### 4. Fully Responsive Design
Using CSS **Media Queries**, the layout automatically adjusts for:
* **Mobile:** Stacked layouts with optimized font sizes for small screens.
* **Tablet:** Increased spacing and adjusted navigation gaps.
* **Desktop:** Full-width professional presentation with a max-width container of 800px for optimal readability.

### 5. Organized Data Presentation
The "Project Showcase" section uses a styled HTML `<table>` with a specific `thead` background to clearly communicate project milestones and progress.
